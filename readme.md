# Philosophiemitschrift - Marco Matouschek

## Allgemein

Diese Mitschrift dient zur Unterstützung der Vorlesungen und Übungen des Studiums 'BA Philosophie' der Universität Wien. Informationen und Zusammenfassungen sollten nicht als perfekte und korrekte Widergabe des gelehrten Inhalts herangezogen werden - sie sind lediglich meine persönliche Auffassung dessen.
